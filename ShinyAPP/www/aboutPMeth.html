Background

DNA methylation is a covalent epigenetic modification involving the enzymatic transfer of a methyl group to specific nucleotides, most commonly cytosine or adenine without altering the primary DNA sequence.The principal forms of DNA methylation include N4-methylcytosine (4mC), 5-methylcytosine (5mC) and N6-methyladenine (6mA), each exhibiting distinct genomic distributions, catalytic pathways, and regulatory functions in cellular processes.

About OpEnCAST

"OpEnCAST" is an optimized ensemble based tool for predicting DNA methylation in plants. It detects different kind of DNA methylation signatures (4mC, 5mC, 6mA) from DNA sequences through an ensemble of machine learning algorithms namely Random Forest, Support Vector Machine along with Gradient Boosting and eXtreme Gradient Boosting based on Monocot (Oryza Sp.) and Dicot (Arabidopsis sp.) reference models respectively.

Instructions for Users

  1. File Format: Provide the input file in .fasta (Fasta file format). Please use valid fasta files where each sequence is 41bp long (For further information please look into "Resource Availability" tab).
  2. Analysis Guidelines: After providing input file (Fasta File), User has to specify the reference model of interest among the provided options viz. Monocot (Oryza Sp.) or Dicot (Arabidopsis sp.) or Both References (Oryza Sp. + Arabidopsis sp.) .Then finally select the "Run Analysis" button for predicting DNA methylation.
